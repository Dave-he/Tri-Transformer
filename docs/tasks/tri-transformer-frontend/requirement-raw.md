# 需求原文 - Tri-Transformer 前端部分

## 来源文档
`docs/Tri-Transformer 可控对话与 RAG 知识库增强系统.md`

## 需求摘要（前端相关部分）

### 产品定位
融合三分支 Transformer 扭合架构与向量化文档知识库（RAG）的高端对话与内容生成系统前端界面。

### 前端技术栈
React + Ant Design Pro

### 前端功能需求（来自 PRD 3.1.4）

#### 对话界面
- 支持单轮/多轮对话
- 实时显示检索知识来源（可追溯）
- 对话历史保存与导出
- 支持文档上传直接提问

#### 可视化管理面板
- RAG 知识库管理界面（文档上传、删除、检索测试）
- 模型训练状态监控
- 性能指标可视化（检索准确率、生成 BLEU 分数、幻觉率）

#### 结果后处理（前端展示）
- 事实一致性校验结果展示
- 格式标准化输出（排版、引用）

### 非功能需求
- 检索速度：< 500ms（Top 10 结果）
- 生成速度：< 2s（1024 token 输出）
- 支持 10+ 并发用户

### 部署
- Docker Compose 一键部署
- 可视化 Web 界面，非技术用户可快速上手

### 路线图第一阶段（MVP，1-2 个月）
- 基础对话界面（单轮对话、文档上传提问）
- 一键部署脚本

### 路线图第二阶段（功能完善，2-3 个月）
- 可视化管理面板
- 支持多格式文档展示
